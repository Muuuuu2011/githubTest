檢查安裝是否成功:

git --version

_______________________________________

第一次設立專案要做的事:

git init 初始化環境
git config --global user.name "您的姓名"
git config --global user.email "您的Email"

_______________________________________

檢查是否設定成功

git config --list

離開環境按小寫q

_______________________________________

前往專案資料夾

cd 資料夾路徑

_______________________________________

資料夾建立檔案後，確認狀態(如果有編輯檔案或新增檔案，就要從這裡再往下循環一次)

git status

_______________________________________

新的檔案會位於Untracked files:
可以使用以下指令開始追蹤檔案

git add 檔案名稱

_______________________________________

追蹤完成後，即可以提交新版本

git commit -m "<填寫版本資訊>"

_______________________________________

完成後可以再次使用git status檢查狀態
或是使用此指令查訊版本

git log
